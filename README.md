# Advent of Code 2022

Résolution des challenges de code de [Advent of Code 2022](https://adventofcode.com/2022) en Ruby.

Les fichiers sources sont dans le répertoire `src/lib` et peuvent être lancés comme ceci :

``ruby nom_du_fichier.rb < donnee_en_entree``

Par exemple, si on veut obtenir la solution du jour 1 et que les données en entrées sont stockées dans un fichier input.txt :

``ruby advent_of_code_1.rb < input.txt``

Des tests unitaires sous **minitest** sont aussi joints.
